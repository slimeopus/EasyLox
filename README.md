# Passdefender - Менеджер паролей

Простой и безопасный менеджер паролей для настольных компьютеров с акцентом на удобство использования.

## Особенности
- Хранение паролей в виде карточек
- Генерация и проверка безопасности паролей
- Поддержка мастер-пароля и шифрования
- Простой и интуитивно понятный интерфейс

## Технологии
- Python 3.x
- Tkinter (для GUI)
- PyCryptodome (для шифрования)
- SQLite (для хранения данных)
