# EasyLox - Менеджер паролей

Простой и безопасный менеджер паролей для настольных компьютеров с акцентом на удобство использования и максимальную защиту персональных данных.

## Особенности
- Хранение паролей в виде карточек
- Генерация и проверка безопасности паролей
- Поддержка мастер-пароля и шифрования (AES-256)
- Простой и интуитивно понятный интерфейс
- Поддержка светлой и тёмной тем оформления
- Поиск и сортировка паролей
- Резервное копирование данных
- Проверка на использование логина или названия сервиса в пароле

## Технологии
- Python 3.x
- Tkinter (для GUI)
- cryptography (для шифрования AES-256)
- JSON (для хранения данных)
